# 🧸 玩偶城

> 面向女性用户的玩偶电商平台，暖粉色主色调，移动端优先的小程序风格设计。

## 📱 预览

直接在手机浏览器中打开 `index.html` 即可体验完整 UI。

### 功能页面

**客户平台（2个页面）**
- 🏠 **首页** — 轮播图 + 分类导航 + 商品瀑布流
- 👤 **我的** — 登录注册 + 个人信息 + 订单管理

**商家平台（3个管理模块）**
- 📦 商品管理 — 编辑商品、上下架、图片上传
- 👥 用户管理 — 查看用户信息与订单
- 📋 订单管理 — 处理订单、发货、退款

### 启动流程

```
打开页面 → 平台选择
  ├─ 客户平台 → 直接进入商城
  └─ 商家平台 → 输入邀请码 → 进入后台
```

**演示邀请码**：`WDC888` / `WD2026` / `MERCHANT`

## 🎨 设计系统

- **主色**：#FF6B81（暖粉色）
- **风格**：微信小程序风格
- **最大宽度**：430px（移动端优先）
- **无外部依赖**：纯 HTML + CSS + JavaScript

## 🚀 部署

### GitHub Pages
1. 将此仓库推送到 GitHub
2. 进入 Settings → Pages
3. Source 选择 `main` 分支
4. 保存后等待部署完成

### Vercel / Netlify
直接导入 GitHub 仓库即可自动部署。
